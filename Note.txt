Compile manually:
# Instruction - .exe file names should be all lowercase
#             - every change within the source code do a save [ctr-s] then do Step 1
Step 1: input in Terminal -> g++ filename.c -o filename.exe 
Step 2: .\filename.exe

For VSCode Compile automatically:
create file named .vscode
inside put the following -
    create settings.json
    copy-paste: 

    {
        "files.associations": {
            "stdlib.h": "c"
        }
    }

    create tasks.json
    then copy-paste:

    {
        "version": "2.0.0",
        "tasks": [
            {
            "label": "build and run C file",
            "type": "shell",
            "command": "gcc",
            "args": [
                "${file}",
                "-o",
                "${fileDirname}/${fileBasenameNoExtension}",
                ";",
                "${fileDirname}/${fileBasenameNoExtension}"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "problemMatcher": ["$gcc"],
            "detail": "Generated task by user."
            }
        ]
    }
